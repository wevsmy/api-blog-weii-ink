# Python: api_blog_weii_ink

A barebones Django app, which can easily be deployed to Heroku.

This is the api used by [my blog](https:blog.weii.ink)

## Running Locally

Make sure you have Python 3.7 [installed locally](http://install.python-guide.org). To push to Heroku, you'll need to install the [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli), as well as [Postgres](https://devcenter.heroku.com/articles/heroku-postgresql#local-setup).

```sh
$ git clone https://github.com/wevsmy/api_blog_weii_ink.git
$ cd api_blog_weii_ink

$ python3 -m venv api_blog_weii_ink
$ pip install -r requirements.txt

$ createdb api_blog_weii_ink

$ python manage.py migrate
$ python manage.py collectstatic

$ heroku local
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```sh
$ heroku create
$ git push heroku master

$ heroku run python manage.py migrate
$ heroku open
```
or
